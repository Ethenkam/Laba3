## Laba3 — Система управления фитнес-клубом (GUI)

Графическая версия системы управления фитнес-клубом из Laba1 с полноценным интерфейсом на PyQt6.

### Особенности

- Полноценный GUI на PyQt6 вместо консольного интерфейса
- Вкладки для управления разными сущностями
- Визуальные формы для добавления/редактирования данных
- Таблицы с фильтрацией и сортировкой
- Отчёты и аналитика

### Структура проекта

```
Laba3/
├── gui_main.py                    # Точка входа для GUI-версии
├── main.py                        # Консольная версия (из Laba1)
├── gui/                           # GUI-компоненты
│   ├── main_window.py            # Главное окно приложения
│   ├── members_tab.py            # Управление участниками
│   ├── coaches_tab.py            # Управление тренерами
│   ├── rooms_tab.py              # Управление залами
│   ├── classes_tab.py            # Групповые занятия
│   ├── plans_tab.py              # Тарифные планы
│   └── reports_tab.py            # Отчёты и аналитика
├── classes/                       # Модели (из Laba1)
│   ├── people.py
│   ├── gym_room.py
│   ├── group_class.py
│   └── ...
└── repositories/                  # Слой данных

```

### Возможности интерфейса

**Участники:**
- Добавление/редактирование участников
- Продление абонементов
- Отслеживание статуса активности
- Просмотр истории платежей

**Тренеры:**
- Управление персоналом
- Указание специализаций
- Назначение на занятия

**Залы и занятия:**
- Создание расписания
- Управление вместимостью
- Запись участников на групповые тренировки

**Отчёты:**
- Список активных абонементов
- Статистика посещений
- Загрузка залов

### Зависимости

```bash
pip install PyQt6
```

Остальные зависимости — стандартные библиотеки Python.

### Запуск

**GUI-версия (основная):**

```bash
python gui_main.py
```

**Консольная версия:**

```bash
python main.py
```
